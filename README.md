# Analytics Backend

Backend service for the User Analytics Application built for the CausalFunnel Full Stack Engineer Assignment.

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Features

* Event Tracking API
* Session Analytics API
* User Journey API
* Heatmap Data API

## API Endpoints

POST /api/events/track

GET /api/events/dashboard

GET /api/events/session/:sessionId

GET /api/events/heatmap?pageUrl=/
